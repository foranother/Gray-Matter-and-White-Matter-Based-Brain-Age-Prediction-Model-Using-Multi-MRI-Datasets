Description

본 프로그램은 뇌 연령 예측을 위해 설계된 3D Transformer 기반 모델로, 회백질과 백질의 구조적 특징을 효과적으로 활용하는 데 독창성을 갖습니다. 뇌의 노화 과정에서 나타나는 회백질과 백질의 변화는 중요한 바이오마커로 작용하며, 이를 통해 뇌 연령을 정밀하게 예측할 수 있습니다. T1 가중 자기공명영상(T1w MRI)을 주요 데이터로 사용하여 높은 해상도의 뇌 구조 정보를 분석하며, 회백질과 백질 정보를 핵심 요소로 통합하여 예측 성능을 극대화합니다.
또한, 성별에 따른 뇌 구조 변화 양상을 반영한 학습 설계를 통해 기존 연구 대비 우수한 성능을 보이며, 국소적인 회백질 상호작용과 전역적인 백질 연결성을 동시에 고려하는 구조를 채택했습니다. 이를 통해 뇌 구조의 변화를 정밀하게 반영하며, 신경퇴행성 질환의 조기 탐지와 노화 연구에 기여할 수 있는 기반을 제공합니다.

Environment

requirements.txt 참조

Files

dataset - mri 데이터셋
model.py - 모델 코드
trained_model.pth - 학습된 model


Usage

0. requirements.txt 참고하여 환경 설정
1. python model.py 를 실행하여 모델 학습 및 저장(결과 포함)
2. python model.py trained_model.pth를 실행하여 저장된 모델 실행 후 결과값 반환
